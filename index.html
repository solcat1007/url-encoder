URL编码解码工具

对文本进行URL编码（Percent-encoding）和解码，支持两种编码模式。

主要功能

  • 编码模式：文本 → URL编码（%xx格式）
  • 解码模式：URL编码 → 原始文本
  • 两种标准：encodeURI（保留URI保留字符）和 encodeURIComponent（严格编码所有特殊字符）
  • 参数解析：一键解析URL查询参数为表格

快捷键

  • Ctrl+Enter 执行转换
  • Ctrl+Shift+X 交换输入输出内容

提示

  • 编码整个URL（含 ? & = 等保留字符）用 encodeURIComponent 模式
  • 仅编码URL参数值用 encodeURI 模式
  • 参数解析功能可快速将 Query String 拆分为键值对
（保留 : / ? # [ ] @ ! $ & ' ( ) * + , ; = 不编码）
