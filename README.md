# URL 编解码

> URL 编码/解码工具，完整 URL 和组件编码，中文百分号编码互转

## 功能概览

- **完整 URL 编码：全部非 ASCII 字符和特殊符号转 %XX**
- **组件编码（encodeURIComponent）：空格转 %20 等严格编码**
- **URL 解码：百分号序列还原为原始字符**
- **中文到百分号编码：直接输入中文即时编码**
- **一键复制结果，交换/清空输入输出面板**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
